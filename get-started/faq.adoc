== Zugriff auf die Lebenszyklusplanung

*Wie lautet die URL zur Lebenszyklusplanung?*

Geben Sie als URL in einem Browser Folgendes ein: https://console.netapp.com/["https://console.netapp.com/"^] um auf die NetApp Console zuzugreifen.



== Lizenzierung

*Benötigen Sie eine Lizenz, um die Lebenszyklusplanung zu nutzen?*

Eine NetApp -Lizenzdatei (NLF) ist nicht erforderlich.
